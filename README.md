PROJECT OVERVIEW
This project focuses on analyzing bank loan applications to understand loan approval trends and credit risk patterns. Using MySQL, the project evaluates how factors such as income, credit score, employment type, and loan amount influence loan approval decisions.

OBJECTIVES
Analyze loan approval vs rejection trends
Identify high-risk and low-risk customers
Study impact of income and credit score on loan decisions
Apply advanced SQL concepts for real-world business analysis

TOOLS AND TECHNOLOGIES
MySQL
MySQL Workbench
SQL (Joins, CASE, Subqueries, Aggregations)

KEY SQL ANALYSIS
Loan approval rate calculation
Credit risk classification using CASE statements
Income band analysis
High-risk customer identification
Above-average loan amount analysis using subqueries


Key INSIGHTS
Customers with higher credit scores have better approval rates
Low-income and low-credit-score customers face higher rejection
Salaried customers show lower credit risk compared to self-employed

CONCLUSION
This project demonstrates practical SQL skills applied to a real-world banking scenario, highlighting data-driven decision-making and risk analysis relevant to Data Analyst roles.
